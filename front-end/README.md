# Tax Intelligence Frontend

Next.js frontend for the Sri Lanka IRD Tax Intelligence & Compliance Assistant.

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Backend

Make sure the FastAPI backend is running on `http://localhost:8000`.

## Features

- Clean, minimal chat interface
- ChatGPT-like message layout
- Real-time responses from the tax intelligence agent
- Dark mode support
- Responsive design
